# reflex_marubatsu_game

## デモを動かす

### 仮想環境を作成し，アクティベートする．

macOS, Linuxの場合

```bash
python -m venv .venv
source .venv/bin/activate
```

Windowsの場合

```powershell
py -3 -m venv .venv
.venv\\Scripts\\activate
```

### その後，以下のコードを実行．

```bash
pip install -r requirements.txt
reflex run
```
